# Returns via Third-Party Courier Drop-Off Points

This document covers using an authorized third-party courier drop-off location to return an online order, for customers who cannot visit a Landmark store or schedule a reverse-pickup. It is a companion to the Returns and Exchange Policy and the Refund Methods and Store Credit Policy; all standard condition and eligibility rules from those documents still apply to a drop-off return, and this document covers only the drop-off-specific mechanics.

## When Drop-Off Is the Right Option

Landmark's standard online return flow offers two collection methods: a scheduled reverse pickup from the customer's address, or a store return at any location of the purchasing brand. Third-party courier drop-off is offered as a third option specifically for customers who live outside a reverse-pickup service area or too far from a Landmark store to visit conveniently, and is presented as a selectable return method in the "My Orders" return-initiation flow only where it is available for the customer's postal code. It is not offered as a substitute for a customer who simply prefers not to wait for a scheduled pickup slot when one is otherwise available.

## Authorized Drop-Off Network

Landmark's return drop-off network is limited to the courier partner's own branded drop-off points (typically the same courier used for the original outbound delivery) — standalone courier service points, and, in the UAE and KSA, select courier-partner kiosks located within participating supermarkets and convenience stores. Drop-off at a general postal counter, a competitor courier's location, or an unaffiliated mall kiosk is not accepted and will not be scanned into Landmark's return system, since only the specific authorized network is integrated with Landmark's reverse-logistics tracking. The nearest authorized drop-off point for a given order is shown within the return-initiation flow once the customer selects "Courier Drop-Off" as the return method, along with that location's operating hours.

## Packaging Requirements

Because a drop-off return is handled by the courier's counter staff rather than a Landmark store associate trained to inspect returns on the spot, packaging requirements are stricter than for a store return:

- The item must be repackaged in its original packaging (box, garment bag, or shoe box) with all original tags and hygiene seals intact, exactly as required for any return, but for drop-off the customer must also affix the pre-paid return label generated by the app or emailed at the time the return was initiated, printed or shown as a scannable digital barcode.
- The package must be sealed by the customer before arriving at the drop-off point; counter staff do not open, inspect, or verify the contents against the original order, since drop-off points are a logistics handoff only, not a returns-inspection service. This means condition and eligibility checks (tags, wear, hygiene seal status) are performed later at the returns processing warehouse, not at the point of drop-off.
- Multiple items from the same order being returned together can be consolidated into a single package under one return label where the original packaging allows, reducing the number of separate drop-offs required.

Because the drop-off counter does not inspect the item, a return later found to fail the standard condition requirements (as set out in the Returns and Exchange Policy) is rejected at the warehouse stage rather than at the counter, and the refund is declined with the item shipped back to the customer at their cost — the same outcome as a failed in-store or pickup return, but discovered later in the process.

## Refund Timeline Compared to Store and Pickup Returns

A drop-off return takes longer to reach the refund-approval stage than a store return, since the item must first be collected from the drop-off point by the courier's regular route, then transported to Landmark's returns processing warehouse for inspection, before the refund is triggered. Customers should expect:

- **Store return**: inspected and refund-triggered immediately at the counter, with the refund following the standard method-based timeline from that point (5-7 business days for cards, and so on).
- **Scheduled reverse pickup**: inspected at the warehouse after courier collection, typically adding 2-4 business days before the refund is triggered, ahead of the standard method-based timeline.
- **Courier drop-off**: typically adds 3-6 business days before the item reaches the warehouse and is inspected, reflecting the drop-off point's normal collection route cadence rather than a dedicated Landmark pickup, ahead of the standard method-based refund timeline.

This means a drop-off return generally takes several days longer overall than a store return before the refund is issued, even though the underlying refund processing timeline once triggered is identical. CX should set this expectation proactively when a customer chooses drop-off, particularly if the customer is returning a time-sensitive purchase (for example, needing the refund before an upcoming payment cycle).

## Proof of Drop-Off

Customers should retain the drop-off receipt or digital confirmation provided by the courier counter at the time of drop-off, since this is the customer's proof that the item was handed over, distinct from Landmark's own tracking, which only updates once the courier's system scans the package into its onward-transport stage. If a drop-off return does not show a tracking update within 5 business days of the customer's drop-off date, the customer should contact CX with the courier's drop-off receipt so a tracing request can be opened with the courier partner, following an investigation approach consistent with the delayed-package process described in the Shipping and Delivery FAQ.
